### 1. **Premium Modern UI/UX**

- **EmailJS Integration**: A fully functional professional contact form that sends inquiries directly to HSN's official email without needing a backend server.

## üõ†Ô∏è Getting Started

### Prerequisites
- Node.js (v18 or higher)
- A valid Gemini API Key
- EmailJS Service/Template IDs

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Zeeshan-Haider-Soomro/hsn-security-services.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file and add your credentials:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   VITE_EMAILJS_SERVICE_ID=your_id
   VITE_EMAILJS_TEMPLATE_ID=your_id
   VITE_EMAILJS_PUBLIC_KEY=your_key
   VITE_WHATSAPP_NUMBER=923218456611
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
